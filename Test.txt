git clone - скачиваем репозиторий с github на компьютер. далее надо перейте в загруженную папку!

git add . - добавляем все файлы в репозиторий.
git add имя.файла -добавляем конкретный файл в репозиторий.

git commit -m "комментарий" - добавляем коментарий о внесенных изминениях.

git push - загружаем изменения на github при первой загрузке требуется ввести логин/пароль.

git status - проверяет статус репозитория
