git clone - скачиваем репозиторий с github на компьютер. далее надо перейте в загруженную папку!

git add . - добавляем все файлы в репозиторий.
git add имя.файла -добавляем конкретный файл в репозиторий.

git commit -m "комментарий" - добавляем коментарий о внесенных изминениях.

git push - загружаем изменения на github при первой загрузке требуется ввести логин/пароль.
!!! перед пушем нужно смаржить локальную ветку с удаленной !!!

git status - проверяет статус репозитория


---------

добавление нового репозитория с компютера

в пакете нужноq программы инициализируем репозиторий git init.

добавляем нужные файлы при помощи git add имя.файла.

коммитим.

и с созданного на github репозитория копируем комманду:
git remote add origin https://github.com/GrySurkov/имя_репо.git
git push -u origin master

Ветвление

git branch имя ветки - создание новой ветки

git branch имя_ветки имя_комита - создать ответвление из коммита

git checkout имя ветки - переход на новую ветку

git checkout имя коммита - переход к нужному коммиту
